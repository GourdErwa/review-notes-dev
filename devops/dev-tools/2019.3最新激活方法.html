# Jetbrains 2019.3 最新激活方法

仅推荐给刚刚入门学习的童鞋们，有经济能力推荐购买正版。如有问题请留言。

# 一、下载激活补丁

链接: 网盘 ，提取码: vh7e

将下载的激活补丁包存在某个文件夹下。

# 二、激活步骤

  1. 打开对应的产品，此处以 IDEA 为例

  2. 编辑 Edit Custom VM Options 文件（参考下面「提示」内容，推荐了 2 种方式）

  3. 在打开的 Edit Custom VM Options 窗口中添加 -javaagent:激活补丁 jar 对应路径

  4. 重启 IDEA

  5. Register... -> 输入激活码（如下）
